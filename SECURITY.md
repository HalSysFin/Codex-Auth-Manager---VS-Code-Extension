# Security Policy

CAM - VS Code Extension handles leased auth material from CAM Auth Manager.

## Sensitive Data

This extension may handle:

- leased auth payloads
- local auth file contents
- manager bearer tokens
- machine and lease identifiers

Do not paste tokens, auth payloads, or full auth files into public issues.

## Reporting a Vulnerability

Please report security issues privately to the maintainer before opening a public issue.
